## Projet paquet Python calculator 

Ce projet est une implémentation capable d'effectuer des opérations mathématiques de base. 
Il a été réalisé dans le cadre d'un TP sur les bonnes pratiques de développement Python.

## Fonctionnalités :

La classe `SimpleCalculator` propose les méthodes suivantes :

`fsum(a, b)`: Addition de deux entiers.
`substract(a, b)`: Soustraction de deux entiers 
`multiply(a, b)`: Multiplication de deux entiers 
`divide(a, b)`: Division de deux entiers

## Paramètres :
- Seuls les types `int` sont acceptés en entrée.
- Gestion des erreurs avec `TypeError` pour les mauvais types et `ZeroDivisionError` pour la division par zéro.
- Il est conseillé d'utiliser un environnement virtuel venv.
